# Projeto de Estudo Java Spring - DsList

# Sobre o projeto

DSlist é um projeto da semana **Intensivão Java Spring**, evento organizado pela [DevSuperior](https://devsuperior.com "Site da DevSuperior").

Um Projeto Spring Rest/Maven com Ambiente Local com Docker compose, Processo de Homolocação local, Deploy CI/CD e configuração CORS.
